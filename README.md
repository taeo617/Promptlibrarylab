# 🫧 Promptlibrary
> **실시간으로 함께 만들고 공유하는 실전 AI 프롬프트 놀이터 & 아카이브**
> 
> Promptlibrary는 누구나 자신만의 AI 프롬프트를 제안하고, 타인과 실시간으로 협업하며 발전시키고, 엄선된 실전 프롬프트를 직관적인 비주얼로 습득할 수 있는 **Apple 감성의 인터랙티브 웹 플랫폼**입니다.

## 💡 왜 Promptlibrary인가요? (프로젝트 기획 의도)

AI 시대에 **"프롬프트(Prompt)"**는 창작의 퀄리티를 결정짓는 핵심 열쇠입니다. 하지만 기존의 수많은 프롬프트 사이트들은 텍스트 위주의 지루한 나열식 구조를 가졌으며, 사용자들이 프롬프트를 함께 실험하고 개선하는 **'실시간 소통과 재미'**가 결핍되어 있었습니다.

Promptlibrary는 이러한 문제를 다음의 세 가지 가치로 해결합니다.

1. **감각적이고 직관적인 비주얼 경험 (Apple-inspired UX)**
   - 딱딱한 리스트를 벗어나, 흐르는 듯한 **액체 유리(Liquid Glass)** 스타일의 라운드 버블이 캔버스 위를 자유롭게 둥둥 떠다닙니다. 마우스나 터치로 버블을 밀쳐내거나 크기를 키울 수 있어, 탐색하는 재미를 극대화했습니다.
2. **집단지성 기반의 실시간 협업 공간 (Real-time Playground)**
   - Firebase Firestore 기반의 초고속 동기화 기술을 적용했습니다. 여러 명의 사용자가 동시에 접속해 프롬프트를 입력하고, 버블을 끌어 옮기고, 텍스트를 수정하면 **모든 사람의 화면에 실시간(1초 미만)으로 즉각 업데이트**됩니다. 좋은 프롬프트를 함께 수정해가며 정교화할 수 있습니다.
3. **눈으로 증명하는 고품질 레퍼런스 아카이브 (Interactive Library)**
   - 검증된 최고 퀄리티의 실전 프롬프트(예: 시네마틱 업스케일링 등)를 엄선하여 보관합니다. 프롬프트 적용 전/후를 볼 수 있는 **인터랙티브 Before/After 슬라이더**를 탑재하여, 프롬프트의 위력을 사용자가 직관적으로 체감하고 원클릭 복사로 즉시 실전에 도입할 수 있게 설계했습니다.

## ✨ 핵심 기능 소개

### 1. 🫧 대시보드 (Floating Canvas)
- **액체 유리(Liquid Glass) 버블 UI**: 사용자가 등록한 프롬프트들이 살아 움직이듯 캔버스 위를 부드러운 애니메이션과 함께 부유합니다.
- **자유로운 공간 배치 & 스케일링**: 버블을 마우스나 터치로 드래그하여 배치할 수 있으며, 버블 우측 하단을 당겨 가로 크기를 손쉽게 늘리거나 줄일 수 있습니다. (너비와 좌표 또한 실시간 저장됩니다.)
- **더블 클릭 & 롱 프레스 편집**: PC 환경의 더블 클릭, 모바일 환경의 롱 프레스 인터랙션으로 버블을 즉시 텍스트 상자로 전환해 현장에서 실시간 협업 편집이 가능합니다.

### 2. 📝 리스트 뷰 (List View)
- **타임라인 타임스탬프**: 실시간 등록되는 프롬프트들을 타임라인 정렬(최신순, 오래된순)로 일목요연하게 한눈에 탐색합니다.
- **나의 작성 글 모아보기 (MY 필터)**: 자신이 썼던 글만 필터링하여 따로 모아보거나, 관리자 모드에서는 실시간 참가자별(이니셜별)로 정밀하게 필터링할 수 있는 기능을 제공합니다.

### 3. 📚 프롬프트 라이브러리 (Prompt Library)
- **검증된 실전 큐레이션**: 단순 장난식이 아닌, 실제 이미지 생성 AI 등에서 압도적인 퀄리티를 만들어내는 전문가급 다각도 프롬프트를 담고 있습니다.
- **인터랙티브 슬라이더**: 마우스 움직임에 반응해 Before(원본)와 After(결과물)를 가르는 슬라이더를 장착하여 프롬프트의 보정/합성 능력을 시각적으로 대조합니다.
- **원클릭 다국어 카피 툴**: 한글(KO) 설명과 대조되는 완벽한 타겟팅 영어(EN) 프롬프트 탭을 별도 제공하여 즉시 딥러닝 모델에 복사-붙여넣기 할 수 있습니다.

### 4. 🔑 심플 계정 & 유연한 권한 시스템
- **초간편 이니셜 로그인**: 복잡한 회원가입 절차 없이 고유 이니셜 3글자(예: `TY`)와 공통 비밀번호만으로 즉시 발급 및 활동이 가능합니다.
- **역할 기반 접근 제어 (RBAC)**:
  - **Guest(게스트)**: 검증된 라이브러리 내 프롬프트를 탐색하고 복사하여 실전에 활용하는 계정 (`guest` / `2026`)
  - **User(사용자)**: 실시간 캔버스에 자유롭게 참여하여 프롬프트 버블을 쏘아 올리고 드래그/편집하며 소통하는 액티브 멤버
  - **Admin(관리자)**: 부적절한 프롬프트의 일괄 청소(Cleanup) 및 라이브러리 프롬프트의 실시간 추가/오버라이딩 편집/삭제 권한 보유 (`admin` / `3913`)
- **비로그인 사용자 보호 UX**: 비로그인 방문자에게는 라이브러리 결과 이미지가 블러(Blur) 처리되며, 복사나 상세 조회를 시도하면 로그인 창으로 안내하는 매끄러운 온보딩 게이트를 구축했습니다.

## 🛠 Tech Stack

프레임워크나 과도한 의존성 라이브러리 없이, 브라우저가 제공하는 원초적 성능을 최대화하는 **경량 & 고효율 아키텍처**로 빌드되었습니다.

- **Frontend Core**: `HTML5`, `Vanilla CSS (Apple Glassmorphism Design System)`, `Vanilla JavaScript (ES6+)`
- **Backend & Database**: `Firebase Firestore (Real-time Synchronization)`, `Firebase Analytics (Usage Tracking)`
- **Fonts & Aesthetics**: `Pretendard (Korean Neo-grotesque Sans-serif)`, `Google Fonts Inter`

## 🚀 시작 가이드 (로컬 테스트)

본 프로젝트는 별도의 컴파일/빌드 과정 없이 웹 브라우저에서 다이렉트로 즉시 구동할 수 있습니다.

1. 본 저장소를 복제하거나 다운로드합니다.
2. `index.html` 파일을 더블 클릭하여 크롬, 사파리 등 현대적인 웹 브라우저에서 실행합니다.
3. 실시간 Firestore 연동이 적용되어 있어 로컬에서도 완벽하게 모든 기능을 테스트해볼 수 있습니다.
4. 더 풍부한 실시간 협업 경험을 얻고 싶다면, **다양한 브라우저 창을 2개 이상 띄워놓고 프롬프트 버블을 드래그**해보세요. 다른 창에서 즉각적으로 연동되어 춤추는 버블을 확인하실 수 있습니다!
